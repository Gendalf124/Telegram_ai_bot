🤖 GPT Telegram Bot (Aiogram + g4f)
Проект представляет собой Telegram-бота на базе библиотеки Aiogram 3, который интегрирует возможности нейросетей через библиотеку g4f (GPT4Free). Бот поддерживает систему регистрации пользователей и ведение истории диалогов с использованием базы данных SQLite.

✨ Основные возможности
🤖 Доступ к DeepSeek: Использование модели deepseek_r1 (через g4f).

📝 Система регистрации: Новые пользователи должны пройти простую процедуру регистрации (имя и пароль) перед использованием бота.

📂 Сохранение данных: Информация о пользователях и история сообщений сохраняются в локальной базе данных registr.db.

💬 Режимы общения: * /bk <запрос> — быстрый одиночный вопрос.

/chat — режим непрерывного диалога (бот помнит контекст).

🧹 Управление историей: Автоматическая очистка истории сообщений в базе данных при завершении чата.

🛠 Стек технологий
Язык: Python 3.10+

Библиотека Telegram: aiogram

Нейросети: g4f (GPT4Free)

База данных: SQLite3 (встроенная библиотека Python)

Асинхронность: asyncio

🚀 Установка и запуск
1. Клонирование репозитория
Bash
git clone https://github.com/ваш-логин/ваш-репозиторий.git
cd ваш-репозиторий
2. Установка зависимостей
Рекомендуется использовать виртуальное окружение:

Bash
pip install aiogram g4f sqlite3
3. Настройка токена
Откройте файл бот.py и замените YOUR_BOT_TOKEN_HERE на ваш токен, полученный от @BotFather:

Python
bot = Bot(token="ВАШ_ТОКЕН")
4. Запуск
Bash
python бот.py
⌨️ Команды бота
/start — Начало работы и проверка регистрации.

/register — Запуск процесса регистрации пользователя.

/help — Список доступных команд.

/bk <текст> — Получить быстрый ответ от ИИ без входа в режим чата.

/chat — Включить режим диалога (бот будет отвечать на каждое ваше сообщение).

/stop — Выйти из режима чата и удалить историю переписки из базы.

📊 База данных
Бот использует две таблицы:

articles — хранит данные пользователей (имя, пароль, Telegram ID).

message_history — хранит историю переписки (ID чата, роль — user/assistant, контент, время).

Для просмотра данных вы можете запустить:

Bash
python registr.py
